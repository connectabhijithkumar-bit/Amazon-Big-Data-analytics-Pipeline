# End-to-End E-Commerce Data Analytics Pipeline (Excel, SQL & Power BI)

Designed and implemented an end-to-end analytics pipeline processing over **1,000,000 transactional e-commerce records**, transforming raw sales data into executive-level business intelligence. The workflow progressed from financial data profiling in **Microsoft Excel**, through high-volume relational database engineering in **MySQL**, and culminated in an interactive **Power BI** dashboard for business decision-making.

## 🛠️ Technical Skill Stack

### Excel
- Data profiling and quality assessment
- Business logic modeling (Sourcing Cost & Platform Fee calculations)
- Pivot Tables, Pivot Charts, and Slicers
- Financial metric validation

### SQL (MySQL)
- High-volume bulk ingestion using `LOAD DATA INFILE`
- Schema optimization through data type modifications
- Common Table Expressions (CTEs)
- Advanced Window Functions (`DENSE_RANK() OVER (PARTITION BY ...)`)
- Aggregation, ranking, and performance optimization

### Power BI
- Interactive executive dashboard development
- DAX measures and calculated metrics
- Cross-filtering visual interactions
- KPI cards, trend analysis, and dynamic metric formatting

## 📈 Key Business Insights

- Aggregated and analyzed **₹10+ billion** in Gross Merchandise Value (GMV) across more than **1,000,000** transaction records.
- Identified **Electronics** as the highest-performing category, contributing **66.2% of total revenue (approximately ₹7 billion)**.
- Measured an average **29% platform markdown** across leading brands including Adidas, Apple, Boat, H&M, and HP using dynamic slicer-driven analysis.
- Built an executive reporting dashboard enabling interactive exploration of revenue, category performance, pricing behavior, and business KPIs.

## 🎯 Outcome

This project demonstrates expertise in large-scale data processing, SQL database engineering, business analytics, and Power BI dashboard development, delivering actionable insights that support data-driven business decisions.
